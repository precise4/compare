# Screengen to compare

```
usage: screengen.py [-h] --source SOURCE [--sname SRCNAME] [--encode ENCODE] [--ename ENCNAME] [--wname WEBNAME] [--web WEB] [--woff WOFF] [--num NUM] [--upload] dir

Generate random screenshots using ScreenGen

positional arguments:
  dir                   Output directory

options:
  -h, --help            show this help message and exit
  --source SOURCE, -s SOURCE
                        Source file
  --sname SRCNAME, -sn SRCNAME
                        Source name
  --encode ENCODE, -e ENCODE
                        Encode file
  --ename ENCNAME, -en ENCNAME
                        Encode name
  --wname WEBNAME, -wn WEBNAME
  --web WEB, -w WEB
  --woff WOFF, -wo WOFF
  --num NUM, -n NUM     Number of screenshots to generate
  --upload, -u          Upload the screenshots to slow.pics
```
